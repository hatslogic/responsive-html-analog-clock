
This is a very simple jQuery based responsive HTML analog clock that can be easily customized for dial colors, needle hand colors, outer borders etc. Its very easy to integrate in your website and can be placed as a widget.


How to use:
----------------------------
Include the javascript - analog-clock-custom.js in the head of the html
Include the css - analog-clock-style.css
Give the class class="analog-clock" to the div where you want to display the clock

Done - clock will show up inside that div!
